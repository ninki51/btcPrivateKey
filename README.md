# btcPrivateKey
碰撞私钥算法：碰撞比特币私钥，破解比特币私钥，破解以太坊私钥的程序
破解比特币私钥和破解以太坊私钥原理：随机生成私钥，然后推导出公钥和地址，然后和丢失私钥的地址比对。
速度每秒生成200万-2亿个私钥。
内带了485万个有币btc地址，可以自行添加。  

靓号生成器GPU版，速度飞快，每秒生成200万以上个地址，支持BTC，ETH，BSC，TRX等主流公链。  

各种密码破解：包括rar密码，pdf密码。
